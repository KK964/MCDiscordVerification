# MCDiscordVerification


MCDiscordVerification is a remake of {TheSourceCode}'s Verification plugin.

  - Custom Commands
  - Account Verification
  - Discord Bot
  - Custom Settings
  - Save Data

# Current Features

  - Custom Commands
  -     /verify (minecraft)
  -     !link (discord)
  - Account Verification
  - Discord Bot
  - Custom Settings
  - Save Data
